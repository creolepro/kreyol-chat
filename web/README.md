# web/

Reserved for the **[kreyol.chat](https://kreyol.chat)** site — a Next.js app where visitors can
talk to and compare the project's Haitian Creole models (see the roadmap in the
[root README](../README.md) and [docs/plan.md](../docs/plan.md)).

Nothing is scaffolded here yet. This directory is a placeholder so the site can be colocated with
the model and corpus work when Phase 4 begins.
