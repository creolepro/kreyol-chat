# fertility/ — Workstream C

Scripts to measure **tokenizer fertility** (tokens per content) for Kreyòl vs English vs French
across ~8 tokenizers — including the first measured Claude / o200k / Llama-3 / Gemma numbers for
Haitian Creole. Outputs a results CSV, a parity-ratio chart, and a one-page report. See
[../../docs/phase-0.md](../../docs/phase-0.md) §Workstream C.
