# tokenizer/ — Workstream B

Scripts to train the **Kreyòl tokenizer v0**: byte-level BPE sweep over {8k, 16k, 24k, 32k} vocab,
whole-word survival evaluation on the core Kreyòl word list, and export (`tokenizer.json` + browser
JSON). See [../../docs/phase-0.md](../../docs/phase-0.md) §Workstream B.
